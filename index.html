Installation

CSS version

  1. Download normalize.css, put it in your /css folder and reference in your html:
  2. Download the all-in-one css minified version, put it in your /css folder and reference in your html:

SASS version

  1. Go to the github repository and download/clone the source code
  2. In the /assets/sass/main.scss file, comment out the style you do not need (you have to keep variables and helpers)
  3. In the /assets/sass/variables.scss file, change the variables to your own design such as color, font size, border radius, etc
  4. Install SASS and compile /assets/sass/main.scss into style.css
  5. Enjoy your style!
